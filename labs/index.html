cabinvis

Visualise binary files in a web application. Inspired by Aldo Cortesi's Visualizing binaries with space-filling curves. Classifies bytes with coloured highlighting, using the Hilbert curve to preserve locality.

charming

Unicode character map web application. Supports characters outside the BMP, lists common encodings, shows basic metadata for each character and allows for searching by codepoint, character and character name.

digilogue

A signal simulator to show to anyone prone to being fooled into buying overpriced cables carrying digital signals.

facepaste

Facebook album downloader for Mozilla Firefox. Saves photos and videos from multiple albums at once, with customisable file naming.

lookout

Lightweight server health monitor using Flask and psutil. Tracks usage of CPU, RAM, swap, disk I/O, network I/O and uptime on Windows and Linux.
