I've seen several quick, web-based configuration testing tools for clients and servers of various protocols, so it could potentially be helpful for me to compile a list of them here. These tools test not only that your servers are working, but that they are configured in accordance with best practices, and I use them all heavily. Let's start with TLS, DNS and SMTP, though I'll try to find more in future and append them here.

DNS — server

  • intoDNS by Elvsoft
  • DNS check tool by Pingdom

SMTP — server

  • DomainKeys, DKIM, SPF, SpamAssassin Email Validator by Brandon Checketts

TLS — server

  • SSL Server Test by Qualys
  • SSL Check by GlobalSign

TLS — client

  • How's My SSL? by Jeff Hodges
  • SSL Client Test by Qualys
